开启个人网站之旅

2024 年 01 月 20 日 5 分钟阅读 Whitefir Studio
web-development personal design

为什么要建立个人网站？

在这个数字时代，拥有一个属于自己的空间来展示作品、分享想法变得越来越重要。作为一名游戏开发者，我希望通过这个网站来展示我的项目、记录开发历程、分享技术心得。

个人网站不仅是一个作品集，更是一个表达自我的平台。在这里，我可以自由地设计每一个细节，打造独特的视觉风格，让访客感受到我的个性和专业性。

技术选型

我选择了简洁的技术栈：Tailwind CSS + 原生 JavaScript。这样可以保持网站的轻量级，同时获得出色的性能表现。

没有选择 React 或 Vue 等框架，是因为对于一个静态内容为主的个人网站来说，原生 JavaScript 已经足够强大。这样做的好处是：

  • 更快的加载速度：没有框架的额外开销，首屏加载极快
  • 更好的 SEO：静态 HTML 对搜索引擎更友好
  • 更简单的部署：可以部署到任何静态托管服务
  • 更低的维护成本：不需要担心框架版本更新和依赖管理

设计理念

遵循 60-30-10 设计原则，60% 纯黑背景营造专业感，30% 玻璃拟态效果增添现代感，10% 强调色点缀关键元素。

这种配色方案不仅视觉上舒适，还能有效引导用户的注意力。纯黑背景让内容更加突出，玻璃拟态效果增加了层次感，而紫色和蓝色的强调色则为页面注入了活力。

玻璃拟态（Glassmorphism）

玻璃拟态是近年来流行的设计趋势，通过半透明背景和毛玻璃模糊效果，创造出类似磨砂玻璃的视觉效果。这种设计风格既现代又优雅，非常适合用于卡片、导航栏等组件。

动画与交互

所有的交互都有即时的视觉反馈。按钮在 hover 时会轻微放大，卡片会有边框高亮效果，页面滚动时元素会淡入并向上滑动。这些细节让网站感觉更加"活"，提升了用户体验。

下一步计划

网站的基础框架已经搭建完成，接下来我计划：

  • 完善项目详情页，添加更多游戏截图和开发故事
  • 实现心情日历功能，记录每天的心情变化
  • 持续更新日志，分享开发经验和技术心得
  • 优化移动端体验，确保在各种设备上都有良好的表现
  • 添加深色/浅色主题切换功能（虽然目前的深色主题已经很棒了）

结语

建立个人网站是一个持续的过程，它会随着我的成长而不断演进。我期待在这个平台上记录更多的故事，分享更多的作品，也希望能够结识更多志同道合的朋友。

如果你也在考虑建立自己的个人网站，我强烈建议你行动起来。这不仅是一个展示自己的机会，更是一个学习和成长的过程。
