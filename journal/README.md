# 日志详情页目录

此目录用于存放个人日志/博客文章的详情页面。

## 文件命名规范

- 使用小写字母和连字符
- 与 `data/journal-entries.json` 中的条目 ID 保持一致
- 示例：`my-first-post.html`

## 页面必备元素

每个日志详情页必须包含：

1. **返回日志列表按钮**：方便用户返回
2. **文章标题**：清晰的标题
3. **发布日期**：显示文章发布时间
4. **文章内容**：完整的文章正文
5. **标签**：文章分类标签（如果有）
6. **阅读时间**：预估阅读时长

## 内容格式

- 支持 HTML 格式的富文本内容
- 标题使用 `<h2>` 和 `<h3>` 标签
- 段落使用 `<p>` 标签
- 代码块使用 `<pre><code>` 标签
- 图片使用 `<img>` 标签，必须包含 `alt` 属性

## 设计规范

- 遵循网站整体的玻璃拟态设计风格
- 正文使用 `leading-relaxed` 确保良好的可读性
- 标题使用 `tracking-tighter` 保持紧凑
- 确保响应式布局
- 最小字体大小 16px

## SEO 优化

- 每个页面都要有独特的 `<title>` 标签
- 包含 `<meta name="description">` 标签（使用文章摘要）
- 添加 Open Graph 标签
- 使用语义化的 HTML 标签
