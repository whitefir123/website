# 项目详情页目录

此目录用于存放各个游戏项目的详情页面。

## 文件命名规范

- 使用小写字母和连字符
- 与 `data/projects.json` 中的项目 ID 保持一致
- 示例：`magic-tower-wars.html`

## 页面必备元素

每个项目详情页必须包含：

1. **返回首页按钮**：浮动按钮，方便用户导航
2. **项目标题**：清晰的项目名称
3. **项目描述**：详细的项目介绍
4. **截图展示**：至少一张游戏截图
5. **技术栈标签**：展示使用的技术和工具
6. **体验按钮**：链接到游戏的在线版本（如果有）

## 设计规范

- 遵循网站整体的玻璃拟态设计风格
- 使用 60-30-10 颜色原则
- 确保响应式布局（移动端友好）
- 所有交互元素都有 hover 效果
- 使用 300ms ease-in-out 过渡动画

## SEO 优化

- 每个页面都要有独特的 `<title>` 标签
- 包含 `<meta name="description">` 标签
- 添加 Open Graph 标签用于社交媒体分享
- 图片必须有 `alt` 属性
