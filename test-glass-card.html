🎨 设计说明

玻璃拟态设计： 采用半透明背景 + 毛玻璃效果 + 微妙边框的组合，创造出现代、高级的视觉效果。

交互反馈： Hover 时边框亮度提升、轻微放大（1.05x）、阴影增强，提供即时的视觉反馈。

性能优化： 使用 CSS transform 而非 position 属性实现动画，确保 GPU 加速和流畅的 60fps。

浏览器兼容： 包含 -webkit- 前缀以支持 Safari 和旧版 Chrome 的 backdrop-filter。
