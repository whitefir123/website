# 图片资源目录

此目录用于存放网站的所有图片资源。

## 图片命名规范

- 使用小写字母和连字符
- 使用描述性的名称
- 示例：`magic-tower-wars-thumb.webp`

## 推荐格式

- 优先使用 `.webp` 格式以获得更好的压缩率和加载速度
- 为不支持 WebP 的浏览器提供 `.jpg` 或 `.png` 后备方案

## 图片类型

### 项目缩略图 (Thumbnails)
- 命名格式：`[project-id]-thumb.webp`
- 推荐尺寸：800x600px
- 用途：项目卡片展示

### 项目截图 (Screenshots)
- 命名格式：`[project-id]-screenshot-[number].webp`
- 推荐尺寸：1920x1080px
- 用途：项目详情页展示

### 其他图片
- 根据用途合理命名
- 保持命名一致性

## 优化建议

1. 压缩图片以减小文件大小
2. 使用适当的分辨率（避免过大）
3. 为移动设备提供响应式图片
4. 使用 `loading="lazy"` 属性延迟加载
